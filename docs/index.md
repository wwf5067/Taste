## 关于这个站点

这是一个个人兴趣知识库，用来记录衣食住行里值得留意的东西，培养并积累自己的兴趣爱好。

当前以**健康生活**为重心，同时维护**香水研究**、**摄影学习**、**情感分析**几个长期板块。后续会逐步扩展「衣 / 食 / 住 / 行」更多方向。
